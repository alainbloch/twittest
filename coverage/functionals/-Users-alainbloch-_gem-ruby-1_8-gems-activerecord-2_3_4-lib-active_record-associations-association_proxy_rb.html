15     # Association proxies in Active Record are middlemen between the object that
 
16     # holds the association, known as the @owner, and the actual associated
 
17     # object, known as the @target. The kind of association any proxy is
 
18     # about is available in @reflection. That's an instance of the class
 
19     # ActiveRecord::Reflection::AssociationReflection.
 
20     #
 
21     # For example, given
 
22     #
 
23     #   class Blog < ActiveRecord::Base
 
24     #     has_many :posts
 
25     #   end
 
26     #
 
27     #   blog = Blog.find(:first)
 
28     #
 
29     # the association proxy in blog.posts has the object in +blog+ as
 
30     # @owner, the collection of its posts as @target, and
 
31     # the @reflection object represents a :has_many macro.
 
32     #
 
33     # This class has most of the basic instance methods removed, and delegates
 
34     # unknown methods to @target via method_missing. As a
 
35     # corner case, it even removes the +class+ method and that's why you get
 
36     #
 
37     #   blog.posts.class # => Array
 
38     #
 
39     # though the object behind blog.posts is not an Array, but an
 
40     # ActiveRecord::Associations::HasManyAssociation.
 
41     #
 
42     # The @target object is not \loaded until needed. For example,
 
43     #
 
44     #   blog.posts.count
 
45     #
 
46     # is computed directly through SQL and does not trigger by itself the
 
47     # instantiation of the actual post records.

226         # Loads the \target if needed and returns it.
 
227         #
 
228         # This method is abstract in the sense that it relies on +find_target+,
 
229         # which is expected to be provided by descendants.
 
230         #
 
231         # If the \target is already \loaded it is just returned. Thus, you can call
 
232         # +load_target+ unconditionally to get the \target.
 
233         #
 
234         # ActiveRecord::RecordNotFound is rescued within the method, and it is
 
235         # not reraised. The proxy is \reset and +nil+ is the return value.
 
236         def load_target
 
237           return nil unless defined?(@loaded)
 
238 
239           if !loaded? and (!@owner.new_record? || foreign_key_present)
 
240             @target = find_target
 
241           end

243           @loaded = true
 
244           @target
 
245         rescue ActiveRecord::RecordNotFound
